# 项目依赖说明

## Web 框架

### fastapi (0.115.0)

现代、高性能的 Python Web 框架，基于 Starlette 和 Pydantic 构建。用于定义 API 路由、请求验证、自动生成 OpenAPI 文档。

### uvicorn[standard] (0.32.0)

ASGI 服务器，用于运行 FastAPI 应用。`[standard]` 额外安装了 `uvloop`（高性能事件循环）、`httptools`（HTTP 解析加速）、`websockets`（WebSocket 支持）等可选依赖。

### python-multipart (0.0.12)

解析 HTTP 请求中的 multipart/form-data 数据。FastAPI 处理文件上传和表单提交时需要此依赖。

---

## 数据验证与配置

### pydantic (2.10.2)

数据验证库，使用 Python 类型注解来定义数据模型。FastAPI 的核心依赖，用于请求/响应体的自动校验、序列化和文档生成。

### pydantic-settings (2.6.1)

Pydantic 的扩展，用于从环境变量、`.env` 文件、命令行参数等来源加载和管理应用配置。

### python-dotenv (1.0.1)

从 `.env` 文件读取键值对并注入到环境变量中。配合 pydantic-settings 使用，方便本地开发时管理敏感配置。

---

## 数据库

### sqlalchemy (2.0.36)

Python 最流行的 ORM（对象关系映射）库。提供 Python 对象与数据库表之间的映射，支持查询构建、连接池管理、事务处理等。

### databases[mysql] (0.9.0)

异步数据库操作库，为 SQLAlchemy 核心查询提供 `async`/`await` 支持。`[mysql]` 额外安装了 `aiomysql` 作为 MySQL 异步驱动。

### pymysql (1.1.1)

纯 Python 实现的 MySQL 客户端驱动。作为 `aiomysql` 的底层依赖，负责与 MySQL 服务器的底层通信。

### cryptography (43.0.3)

加密库，提供各类密码学算法。在使用 SQLAlchemy 连接 MySQL 等数据库时，用于处理 SSL/TLS 加密连接等安全功能。

---

## Redis

### redis (5.2.0)

同步 Redis 客户端库。提供连接池、管道、发布/订阅、Lua 脚本等完整的 Redis 功能支持。

### aioredis (2.0.1)

异步 Redis 客户端。为 Redis 操作提供 `async`/`await` 接口，适用于 FastAPI 的异步上下文。

> **注意**：`aioredis` 于 2021 年停止独立维护，其功能已合并到 `redis-py` 4.0+ 的 `redis.asyncio` 模块中。当前保留此依赖仅为兼容已有代码，后续可迁移到 `redis.asyncio`。

---

## 认证与安全

### passlib[bcrypt] (1.7.4)

密码哈希库，支持多种密码散列算法。`[bcrypt]` 额外安装了 bcrypt 算法实现。用于用户密码的安全存储和验证。

> **注意**：passlib 自 2020 年起未再更新，长期来看可考虑替换为直接使用 `bcrypt` 库。

---

## 工具

### python-dateutil (2.9.0.post0)

日期时间处理扩展库，提供了比标准库 `datetime` 更强大的日期解析、相对时间计算、时区处理等功能。
