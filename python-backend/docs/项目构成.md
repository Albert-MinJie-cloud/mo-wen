## 项目目录架构

```plain
python-backend/
├── .env.example              # 配置文件示例（提交到仓库）
├── .gitignore
├── pyproject.toml            # 依赖管理
├── uv.lock                   # 依赖锁定文件
└── app/
    ├── main.py               # 应用入口
    ├── config.py             # 配置管理
    ├── database.py           # 数据库连接
    ├── deps.py               # 依赖注入（权限控制）
    ├── exceptions.py         # 异常和错误码
    ├── models/               # SQLAlchemy ORM 模型
    │   └── user.py
    ├── routers/              # 路由层（接口定义）
    │   ├── health.py
    │   └── user.py
    ├── schemas/              # Pydantic 模型（请求/响应）
    │   ├── common.py
    │   └── user.py
    ├── services/             # 业务逻辑层
    │   └── user_service.py
    └── utils/                # 工具类
        ├── password.py       # 密码加密
        └── session.py        # Session 管理
```
